## 專案說明 
Demo Website:https://zaihello.github.io/2025-kidshop/#/shop

## 使用技術 
* 以 Vue 3 和 Vite架設開發 
* 使用 tailwindcss v3 作為 CSS 框架，以進行切版 
* Mokky 伺服器 RESTApi

## 使用套件 
* Vue 3 
* Vue Router 
* Vue Axios 
* Vee Validation 
* Tailwindcss v3 
* Swiper 
* Pinia 
* Sweetalert2
