## 使用技術

使用 tailwindcss v3 作為 CSS 框架，以進行切版 
Mokky 伺服器 RESTApi

## 使用套件 
Vue 3 
Vue Router 
Vue Axios 
Vee Validation 
Tailwindcss v3 
Swiper 
Pinia 
Sweetalert2
